class HypixelAPI

export class HypixelAPI;

The main api interacting with the Hypixel API

Constructor

constructor(apiKey: string)

apiKey: The Hypixel Application / Development API Key ⚠️ Development API Key is unrecommended. Create a permanent client application instead.


Methods

⚠️ All of the methods are not static. A valid API key is required to get data.


Example

const { HypixelAPI } = require("../src/index");
const client = new HypixelAPI("YOUR_HYPIXEL_API_KEY_GOES_HERE");
(async() => {
 const data = await client.GetPlayerData("b3e9ab8c9aca4ea6b51785d66a8d3dc7");
 console.log(data);
})();


GetPlayerData(uuid: string)

Obtains user's hypixel data

GetPlayerRecentGames(uuid: string)

Obtains user's recently played games

Private async FetchHypixelAPIEndpoint(endpoint: string, extraData?: {}, requireApiKey?: boolean)

Fetchs the hypixel API
endpoint -> string after /v2/, replace "/" with "."
extraData -> any extra data that is required
requireApiKey -> whether or not the api key is needed to place in header. Default: true

Private CheckHypixelAPIDataStatus(res: Response)

Checks the Fetch respond information, throw error if status is not 200

Private async TestApiKey()

Checks apiKey obtained from constructor. Throws error if hypixel api return status other than 200.
